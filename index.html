Detecting Hematological Disorders Using Machine Learning

Role: Thesis Researcher & Software Developer

A machine learning–based diagnostic support system designed to assist in the detection of hematological disorders, particularly anemia, through automated analysis of blood smear images.

Objectives:

  • Automate blood smear image analysis
  • Assist in early detection of hematological disorders
  • Reduce manual workload and subjectivity in microscopic analysis

Key Features:

  • Blood smear image preprocessing and enhancement
  • Red blood cell (RBC) segmentation and isolation
  • Automated RBC counting using hemocytometer images
  • CNN-based classification of hematological conditions
  • Model evaluation using accuracy and validation metrics

Methodology:

  • Image acquisition from microscope and public datasets
  • Preprocessing using OpenCV techniques
  • Feature extraction from RBC morphology
  • Training and validation of CNN models

Tools & Technologies Used:

Python, OpenCV, TensorFlow, Keras, NumPy, Convolutional Neural Networks (CNN), Image Processing

This undergraduate thesis demonstrates the practical application of machine learning and computer vision in healthcare diagnostics.
